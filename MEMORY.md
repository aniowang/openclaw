## 🛠️ 技術開發與工具經驗 (Technical Insights)

### GitHub 整合解決方案
*   **環境限制**：在部分執行環境中，`git` 或 `gh` CLI 可能不在 PATH 中，導致指令執行失敗。
*   **推薦方法**：優先使用 **GitHub REST API** 進行操作。
*   **操作要點**：
    *   **端點**：使用 `PUT /repos/{owner}/{repo}/contents/{path}` 上傳檔案。
    *   **編碼**：檔案內容必須經過 Base64 編碼。
    *   **權限**：使用的 Token 必須具備 `repo` 範圍權限。
    *   **成功指標**：API 應回傳 HTTP 201 狀態碼。
*   **參考案例**：已在 `mcp_test` repo 完成測試（檔案：`method_api.txt`），相關操作指南已同步至 `SKILL.md`。
